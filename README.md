**Tic Tac Toe (C# WPF)**
Project Overview

This is a simple Tic Tac Toe game built with C# and WPF (Windows Presentation Foundation). The game allows two players to compete on a 3x3 board, taking turns until one wins or the game ends in a draw.

**Features**

Two-player Tic Tac Toe

Graphical interface using WPF

Detects win conditions and draws

Clean object-oriented structure with separate classes for board state, players, and game logic

**Technologies**

C# (Windows Desktop Application)

WPF (Windows Presentation Foundation) for GUI

.NET Framework

**How to Run**

Open the solution file TicTacToe.sln in Visual Studio.

Build the project.

Run the application.

The game window will launch, and players can immediately start playing.
